# AI Media Hub - Shared Types

TypeScript type definitions and constants shared between client and server.

## Usage

Import types in your project:

```typescript
import { MediaItem, User, ContentAnalysis } from '../shared/types';
```

## Type Definitions

### MediaItem
Represents a piece of media content (image, video, article, podcast)

### User
User account information and preferences

### ContentAnalysis
AI-powered analysis metadata for content

### SearchResult
Paginated search results structure

## Constants

### CATEGORIES
Available content categories
